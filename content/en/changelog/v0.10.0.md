![It's happening](http://i.imgur.com/7drHiqr.gif)

v0.10.0 drops support for Rust 1.14 and earlier, and adds support for Rust 1.15 (ON STABLE). `diesel_codegen_syntex` has been removed, and is no longer supported.

Additionally, this release adds initial support for the JSON data type in PostgreSQL. There is also a new `print-schema` subcommand in Diesel CLI which will show you the code generated by `infer_schema!()`. As always, you can find a full list of what has changed in [the changelog](https://github.com/diesel-rs/diesel/blob/v0.10.0/CHANGELOG.md).
